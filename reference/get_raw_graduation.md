# Download raw graduation data from DESE Socrata API

Downloads graduation rate data from the Massachusetts
Education-to-Career Research and Data Hub (Socrata) API.

## Usage

``` r
get_raw_graduation(end_year)
```

## Arguments

- end_year:

  School year end (2023-24 = 2024). Valid years: 2006-2024.

## Value

Data frame with graduation data including district and school records
