# Package index

## Fetch Enrollment Data

Functions for downloading enrollment data from DESE

- [`fetch_enr()`](https://almartin82.github.io/maschooldata/reference/fetch_enr.md)
  : Fetch Massachusetts enrollment data
- [`fetch_enr_multi()`](https://almartin82.github.io/maschooldata/reference/fetch_enr_multi.md)
  : Fetch enrollment data for multiple years
- [`get_available_years()`](https://almartin82.github.io/maschooldata/reference/get_available_years.md)
  : Get available years for Massachusetts enrollment data

## Transform Data

Functions for transforming and summarizing enrollment data

- [`tidy_enr()`](https://almartin82.github.io/maschooldata/reference/tidy_enr.md)
  : Tidy enrollment data
- [`id_enr_aggs()`](https://almartin82.github.io/maschooldata/reference/id_enr_aggs.md)
  : Identify enrollment aggregation levels
- [`enr_grade_aggs()`](https://almartin82.github.io/maschooldata/reference/enr_grade_aggs.md)
  : Custom Enrollment Grade Level Aggregates

## School Classification

Functions for identifying school types (charter, etc.)

- [`get_charter_codes()`](https://almartin82.github.io/maschooldata/reference/get_charter_codes.md)
  : Get charter school codes
- [`get_charter_district_codes()`](https://almartin82.github.io/maschooldata/reference/get_charter_district_codes.md)
  : Get charter district codes

## Caching

Functions for managing local data cache

- [`cache_status()`](https://almartin82.github.io/maschooldata/reference/cache_status.md)
  : Show cache status
- [`clear_cache()`](https://almartin82.github.io/maschooldata/reference/clear_cache.md)
  : Clear the maschooldata cache
