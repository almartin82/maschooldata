# Download raw enrollment data from DESE Socrata API

Downloads enrollment data from the Massachusetts Education-to-Career
Research and Data Hub (Socrata) API.

## Usage

``` r
get_raw_enr_api(end_year)
```

## Arguments

- end_year:

  School year end (2023-24 = 2024). Valid years: 1994-2025.

## Value

Data frame with enrollment data including district and school records
